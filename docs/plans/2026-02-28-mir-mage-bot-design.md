# 热血传奇法师自动练级外挂 — 设计文档

## 概述

为 996 引擎传奇私服开发一个 Windows 端法师自动练级工具。采用 Python + 图像识别 + 模拟操作方案，不读写游戏内存、不发封包，通过截屏分析游戏状态并模拟键鼠操作。

## 目标

- 法师角色全自动挂机练级
- 支持两种练级模式：火力输出模式、宝宝练级模式
- 智能吃药、回城补给、自动买药、自动寻路
- GUI 控制面板，方便配置和监控

## 技术选型

| 组件 | 库 | 说明 |
|------|-----|------|
| 截屏 | `win32gui` + `PIL` | 后台截图，不要求窗口前台 |
| 图像识别 | `OpenCV` | 模板匹配 |
| OCR | `PaddleOCR` | 怪物名称识别（中文效果好） |
| 键鼠模拟 | `win32api` / `ctypes` | 发送消息到窗口，支持后台操作 |
| GUI | `PyQt5` | 控制面板 |
| 配置 | `pyyaml` | 用户配置持久化 |

### 核心模块

1. **屏幕捕获 (Screen Capture)** — 快速截取游戏窗口画面，支持后台截图
2. **图像识别 (Vision Engine)** — OCR 识别怪物名称，颜色比例法识别血蓝条
3. **状态管理 (State)** — 维护角色当前状态（HP/MP/坐标/背包/地图）
4. **AI 决策 (Strategy)** — 基于状态的有限状态机，决定下一步行动
5. **操作执行 (Action Engine)** — 将决策转化为键鼠操作序列
6. **GUI 控制面板** — PyQt5 界面，用于配置和监控

## 识别方案

### 血蓝条识别 — 颜色比例法

- 截取血条区域 → 统计红色像素占比 → HP 百分比
- 截取蓝条区域 → 统计蓝色像素占比 → MP 百分比

### 怪物识别 — OCR

- 使用 PaddleOCR 识别游戏画面中的文字
- 过滤出怪物头顶的名称
- 根据名称判断怪物类型（普通/精英/Boss）
- 根据文字位置推算怪物在屏幕上的坐标

## 练级模式

### 模式一：火力输出模式

法师自己用技能打怪。

**状态机流程：**

1. **巡逻** → 沿路径点移动寻找怪物
2. **发现怪物** → 进入战斗
3. **战斗中** → 根据怪物数量选择技能
   - 单怪：雷电术 / 大火球
   - 多怪：地狱火 / 火墙 + 风筝走位
   - 危险：抗拒火环 → 跑开 → 吃药
4. **怪物死亡** → 捡物品 → 回到巡逻
5. **补给检查** → 药品不足或背包满 → 回城补给 → 飞回练级点

### 模式二：宝宝练级模式

依靠召唤宝宝输出，角色负责引怪和躲避。

**状态机流程：**

1. **检查宝宝** → 无宝宝则召唤
2. **引怪** → 跑向怪物群 → 吸引仇恨 → 跑回宝宝附近
3. **躲避** → 远离怪物保持安全距离 → 让宝宝输出
4. **怪物死亡** → 捡物品 → 回到引怪阶段
5. **异常处理** → 宝宝消失重新召唤 / HP 低吃药 + 火环 / 背包满回城

### 法师技能映射

| 场景 | 技能 |
|------|------|
| 单怪 | 雷电术 / 大火球 |
| 多怪 | 地狱火 / 火墙 |
| 危险 | 抗拒火环 |
| Boss | 冰咆哮 + 风筝 |
| 召唤 | 召唤神兽 |

## 可配置参数

- 吃红药阈值：HP < 50%（默认）
- 吃蓝药阈值：MP < 30%（默认）
- 回城补给阈值：红药 < 5 个
- 练级地图和路径点坐标
- 攻击范围和安全距离
- 捡物品过滤规则（按名称/颜色）
- 技能快捷键映射
- 引怪数量（宝宝模式）
- 练级模式选择

## 开发优先级

1. **P0 — 核心框架**：截屏、窗口管理、键鼠模拟、基础 GUI
2. **P1 — 识别能力**：血蓝条识别、OCR 怪物识别
3. **P2 — 火力输出模式**：基础战斗 + 吃药 + 巡逻
4. **P3 — 宝宝练级模式**：召唤 + 引怪 + 躲避
5. **P4 — 智能补给**：回城买药、背包管理
6. **P5 — 寻路系统**：自动走位、地图切换
7. **P6 — 完善 GUI**：配置面板、日志、状态监控
